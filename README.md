# Farecalc
A simple console-based Fare Calculation System called FareCalc Travel Optimizer built using Python. It calculates ride fares for a ride-sharing service based on distance, vehicle type, and time-based surge pricing. The system dynamically adjusts prices during peak hours and provides a final ride estimate with proper validation.

#Features:
Calculate fare based on distance and vehicle type
Use dictionary mapping for vehicle rates (Economy, Premium, SUV)
Apply surge pricing (1.5x) during peak hours (5 PM – 8 PM)
Handle invalid vehicle types with error messages
Function-based implementation for clean logic
User-friendly formatted price receipt
